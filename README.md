# 🤖 TikTok Video Bot — Інструкція

## Що потрібно встановити

### 1. Python
Завантаж з https://python.org (версія 3.10+)
При встановленні постав галочку "Add to PATH"

### 2. FFmpeg
Завантаж з https://ffmpeg.org/download.html
- Windows: розпакуй і додай папку `bin` до PATH
- Або просто поклади `ffmpeg.exe` в папку з ботом

### 3. Бібліотеки Python
Відкрий термінал в папці з ботом і виконай:
```
pip install -r requirements.txt
```

---

## Налаштування бота

### Крок 1 — Отримай токен бота
1. Відкрий Telegram, знайди @BotFather
2. Напиши /newbot
3. Дай назву боту
4. Скопіюй токен (виглядає як: 1234567890:AAFxxx...)

### Крок 2 — Встав токен в bot.py
Відкрий `bot.py` і заміни:
```
BOT_TOKEN = "ТВІЙ_ТОКЕН_ТУТ"
```
на свій токен.

---

## Запуск

```
python bot.py
```

Бот запустився! Тепер в Telegram знайди свого бота і надішли /start

---

## Як користуватись

1. Відкрий бота в Telegram
2. Надішли відео (до 100МБ)
3. Чекай 1-3 хвилини
4. Отримай 6 унікальних варіантів

## Проблеми?

**FFmpeg не знайдено:**
Постав ffmpeg.exe в ту ж папку що і bot.py

**Помилка pip install:**
Спробуй: `pip install python-telegram-bot==20.7`

**Бот не відповідає:**
Перевір токен в bot.py
